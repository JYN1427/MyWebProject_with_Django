模板语法的学习

所有名字：{{ names }}
按照下标：
{{ names.0 }}
{{ names.1 }}
{{ names.2 }}
列表： {% for name in names %} {{ name }} {% endfor %}
字典：
    {% for k,v in mydict.items %}
  • {{ k }}: {{ v }}
    • {% endfor %} {% for v in mydict.values %}
  • {{ v }}
    • {% endfor %}

    {% for item in datas %}
